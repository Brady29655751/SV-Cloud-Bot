#Features
0. help 幫助
1. battle 對戰開始
2. dice 骰子
3. player 查看對戰資訊
4. deck 查看牌堆資訊
5. keep 起手留牌
6. draw 抽牌
7. search 檢索
8. explore 探索（查看牌堆頂部N張卡片）
9. add 塞進牌堆
10. substitute 置換牌堆
11. effect 新增牌堆卡片資訊
12. save 手動存檔
13. quit 結束對戰
