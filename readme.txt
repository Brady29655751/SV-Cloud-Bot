#Features
0. help 幫助
1. battle 對戰開始
2. 2pick 模式2pick
3. portal 查詢卡片資料
4. travel 漫遊（由符合指定條件的卡片之中隨機顯示1張）
5. filter 過濾（搜尋所有符合指定條件的卡片）
6. dice 骰子
7. player 查看對戰資訊
8. deck 查看牌堆資訊
9. choose 選擇
10. shuffle 洗牌
11. keep 起手留牌
12. draw 抽牌
13. search 檢索
14. explore 探索（查看牌堆頂部N張卡片）
15. add 塞進牌堆
16. substitute 置換牌堆
17. effect 修改牌堆卡片資訊
18. cheat 隨機事件
19. save 手動存檔
20. quit 結束對戰
