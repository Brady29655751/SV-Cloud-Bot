#Features
0. help 幫助
1. battle 對戰開始
2. 2pick 模式2pick
3. portal 查詢卡片資料
4. dice 骰子
5. player 查看對戰資訊
6. deck 查看牌堆資訊
7. choose 選擇
8. keep 起手留牌
9. draw 抽牌
10. search 檢索
11. explore 探索（查看牌堆頂部N張卡片）
12. add 塞進牌堆
13. substitute 置換牌堆
14. effect 新增牌堆卡片資訊
15. save 手動存檔
16. quit 結束對戰
